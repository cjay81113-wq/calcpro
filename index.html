📚 Math & Finance Formulas
Finance
EMI
EMI = P x r x (1+r)^n / ((1+r)^n - 1)
P=Principal, r=monthly rate, n=months
Compound Interest
A = P(1 + r/n)^(nt)
P=Principal, r=annual rate, n=times/year, t=years
Simple Interest
SI = (P x R x T) / 100
P=Principal, R=Rate%, T=Time in years
SIP Returns
M = P x [(1+r)^n - 1] / r x (1+r)
P=monthly investment, r=monthly rate, n=months
ROI
ROI = ((Final - Initial) / Initial) x 100
Percentage
Percentage
% = (Part / Whole) x 100
% Change
% Change = ((New - Old) / Old) x 100
Positive=increase, Negative=decrease
Profit %
Profit% = ((SP - CP) / CP) x 100
SP=Selling Price, CP=Cost Price
GST (Exclusive)
Total = Amount x (1 + GST%/100)
Geometry
Circle Area
A = pi x r^2
Triangle Area
A = (1/2) x base x height
Cylinder Volume
V = pi x r^2 x h
Pythagorean Theorem
c^2 = a^2 + b^2
c=hypotenuse of right triangle
Health
BMI
BMI = Weight(kg) / Height(m)^2
Normal: 18.5 - 24.9
BMR (Mifflin-St Jeor)
Male: 10W + 6.25H - 5A + 5 | Female: 10W + 6.25H - 5A - 161
W=weight kg, H=height cm, A=age years
